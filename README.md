# Dashcam
 Dashcam realizzata con raspberry
 Grazie a piCam registo in tempo reale su file h24.
 Sul video sovrappongo i dati di posizione in tempo reale, rilevati da apposito sensore GPS.
 Vado poi a salvare i video eliminandoli autonomamente dopo 2 giorni di registrazione.
 I dati di velocita' vengono inviati e visualizzati sulla piattaforma di thingspeak
